# Android Voice Assistant

Home Assistant custom integration for an Android-based local voice assistant client.

## Features
- WebSocket-based device registration
- Per-device sensors, switches, numbers, selects, texts, buttons and media player
- Config flow and options flow
- Push-style config updates from Home Assistant to the Android app
- HACS-ready metadata

## Installation
1. Copy `custom_components/android_voice_assistant` into your Home Assistant config folder.
2. Restart Home Assistant.
3. Add the integration via **Settings -> Devices & Services -> Add Integration**.
4. Configure the Android app with your HA URL and long-lived access token.
